gadgIteration hardware v2.1
=============

###digital pinouts###

* D2  - Button "MODE"
* D4  - Button "RANGE"
* D6  - LED Strip
* D7  - MODE LEDs (high=right ON, low=left ON)
* D8  - speaker/audio output
* D13 - onboard green LED

###analog pinouts###

* A0 - Standard Alligator In
* A1 - Human Circuit In
* A2 - SENSOR 1
* A3 - SENSOR 2
* A4 - SENSOR 3
